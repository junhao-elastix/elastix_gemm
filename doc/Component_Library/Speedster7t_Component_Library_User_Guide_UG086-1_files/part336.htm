‌Usage and Inference

ACX_FP_MULT_ADD cannot be inferred and must be directly instantiated. The specified floating point format applies to the inputs and output but, internally, the operations are performed with fp24.

If fp_size=24, the four data inputs require 96 bits total. Since this is more than 72 bits, the ACX_MLP72 (see page 95) that performs the operation is used in wide input mode. In this mode, the adjacent ACX_BRAM72K (see page 248) site is used as route-through, meaning it is no longer available for BRAM placement. By contrast, if fp_size=16, only 64 input bits are needed, and normal input mode is used.
