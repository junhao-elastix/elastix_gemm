‌ACX_NAP_HORIZONTAL‌

The ACX_NAP_HORIZONTAL component provides for 288-bit bidirectional data transport. When a word of data is presented to the transmission interface along with a destination ID, the data and all of the fields are captured and are sent to the destination node at the column indicated by the tx_dest[3:0] signal, which then presents it to the FPGA logic using the destination NAP's receiver interface.
