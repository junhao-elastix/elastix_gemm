‌ACX_FP_MULT_2X‌

The ACX_FP_MULT_2X module is similar to ACX_FP_MULT (see page 217), but uses a single ACX_MLP72 (see page 95) to compute two products in parallel, with optional accumulations. The two operations are:

dout_ab = i_din_a × i_din_b dout_cd = i_din_c × i_din_d
