‌ACX_NAP_ETHERNET‌

The ACX_NAP_ETHERNET component is used to provide Ethernet packet transmission and reception. The NAP is placed on one of the specific Ethernet columns (see the "Speedster7t Ethernet System Architecture" chapter in the Speedster7t Ethernet User Guide (UG097) for details). The NAP then transmits and receives packets from the Ethernet Interface Unit (EIU) within the Ethernet subsystem.
