‌Length Adjustment

The ACX_MLP72 supports programmable end locations for both the write and read address generators. These

thresholds are set using the lram_fifo_wrptr_maxval and lram_fifo_rdptr_maxval parameters.

When an address pointer is equal to the specified maxval threshold, the next increment assigns the address counter back to 0.
