‌LRAM Virtual Ports‌

When the LRAM is configured within the ACX_MLP72, several of the ACX_MLP72 ports are re-purposed to the LRAM. These configurations are also dependent upon the operating mode. These re-purposed ports have logical internal signal names and can be considered virtual ports to the LRAM. The mapping of these virtual ports is detailed below.
