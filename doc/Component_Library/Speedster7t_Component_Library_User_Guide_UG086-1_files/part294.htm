‌Usage and Inference

ACX_INT_COMPLEX_MULT is more efficient than simply writing the complex product in terms of the coefficients, because it maps the operation to a single ACX_MLP72, and each coefficient must be input only once.

When accumulation is enabled, the internal accumulators are 48 bits wide. However, each output coefficient is limited to the 36 least significant bits of the accumulator result.
