‌Standalone LRAM

Note

When an ACX_LRAM2K is instantiated directly, the associated ACX_MLP72 is not available due to the use of shared pins.

If an LRAM independent of the MLP72 is required, use the dedicated ACX_LRAM2K_SDP (see page 280) or ACX_LRAM2K_FIFO (see page 272) primitive, appropriate to the application. These primitives have only the required ACX_LRAM2K ports and parameters, simplifying instantiation.
