"clocked"," unclocked"


"clocked"


Controls whether the assertion of the reset of the input and output registers is synchronous or asynchronous with respect to the clk input:

"clocked" – synchronous reset. The register is reset upon the next rising edge of the clock when the associated rstn signal is asserted low.

"unclocked" – asynchronous reset. The register is reset immediately when the associated rstn signal is asserted low.
