‌ACX_INT_RLB_MULT‌

The ACX_INT_RLB_MULT module implements integer multiplication with fabric logic. Both inputs must be signed, but may have different sizes. This macro features:

N × M multiplication, for N, M = 3–9 Optional accumulator

Optional registers to enable higher frequency Implemented with RLB fabric logic only
