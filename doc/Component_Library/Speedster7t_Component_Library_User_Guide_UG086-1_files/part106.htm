‌ACX_CLKGATE (Clock Gate)‌


Figure 30: ACX_CLKGATE Logic Symbol

Note

When simulating the ACX_CLKGATE component, if the transition on the input signal en and the transition on the input clock arrive at the same moment, the time that it takes for the en transition to have an effect is dependent on the simulator's scheduling of events and may vary with different simulators, different designs, and different simulation models.

The ACX_CLKGATE component implements a clock gate that allows the output to toggle only when the input en is asserted high. The ACX_CLKGATE component disables the clock only after the clock input has transitioned low, guaranteeing that the output is glitchless. The output clock is guaranteed to never have a pulse width narrower in time than the input pulse width.
