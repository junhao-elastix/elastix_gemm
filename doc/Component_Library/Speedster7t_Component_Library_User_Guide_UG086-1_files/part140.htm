‌Numerical Formats‌

The ACX_MLP72 can process the following numerical formats:

Table 99: ACX_MLP72 Supported Numerical Formats



Formats

Integer

int3, int4, int6, int7, int8, int16

Block floating point

BFP Int3, BFP Int4, BFP Int6, BFP Int7, BFP Int8, BFP Int16

Floating point

fp3, fp4, fp6, fp8, fp16, fp16e8, fp24.


See Speedster7t MLP Number Formats for details of each of the numerical formats
