‌Mode 2 Pointer Reset

In Mode 2 (requirement that lram_fifo_enable = 1'b1) two external pins are re-purposed as internal FIFO address generator resets:

Asserting lram_fsm_wrrst resets the FIFO write pointer to 0 on the next active edge of lram_wrclk. Asserting lram_fsm_rdrst resets the FIFO read pointer to 0 on the next active edge of lram_rdclk.

These additional signals allow the read or write pointers to be dynamically reset.
