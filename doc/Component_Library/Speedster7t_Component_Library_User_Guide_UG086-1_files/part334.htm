Table 201: ACX_FP_MULT_ADD Parameters



Parameter

Supported Values


Default


Description

fp_size

16, 24

16

Width of floating-point number. Supports fp24, fp16, and fp16e8.

fp_exp_size

5, 8

5

Size of floating-point exponent.


subtract


0, 1


0

0 – compute (i_din_a × i_din_b) + (i_din_c × i_din_d).

1 – compute (i_din_a × i_din_b) - (i_din_c × i_din_d).


accumulate


0, 1


0

0 – no accumulation: dout = (i_din_a × i_din_b) ± (i_din_c × i_din_d).

1 – accumulation: dout is the accumulated value. The start of accumulation is signaled by asserting i_load=1.


in_reg_enable


0, 1


0

0 – no input registers.

1 – i_din_a, i_din_b, i_din_c and i_din_d are registered.

The input registers are controlled by the i_in_reg_ac_ce, i_in_reg_bd_ce

and i_in_reg_rstn inputs. Enabling the input registers adds one cycle of latency.


pipeline_regs


0–5


0

The number of pipeline registers not counting the input register. The total latency is

pipeline_regs + in_reg_enable.
