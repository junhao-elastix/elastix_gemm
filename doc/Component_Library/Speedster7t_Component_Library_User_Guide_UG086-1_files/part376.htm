‌Advanced Modes‌

The ACX_BRAM72K_SDP supports two advanced modes that allow for remapping of the address space within the memory to be accessed when in 8-bit byte mode and, additionally, for control of the tightly-coupled LRAM within the ACX_MLP72, (see ACX_MLP72 LRAM).

The advanced modes are enabled in the read and write sides by asserting the wrmsel and rdmsel inputs respectively. When asserted, wrmsel and rdmsel are combined with wraddr[11] and rdaddr[11] respectively to configure the write and read side advanced mode.
