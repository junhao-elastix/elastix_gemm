If Pipeline registers is enabled (greater than 1), i_pipeline_rstn is the synchronous active-low reset for the pipeline and accumulator registers.

i_load


Input

If Accumulation is enabled, i_load resets the accumulator to:

(i_din_a × i_din_b) ± (i_din_c × i_din_d) , ignoring the previous value. This signal is internally pipelined to have the same latency as: (i_din_a × i_din_b) ± (i_din_c × i_din_d) .

If Accumulation is not enabled, i_load does not exist.

2'b11 – last operation had underflow. The result is ± 0.0.
