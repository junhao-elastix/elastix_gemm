‌Simulation‌

o_status

Currently, in full RTL simulation mode, the ACX_DEVICE_MANAGER simply waits a few cycles, then sets

0x3

to to indicate successful completion of the startup phase. This is appropriate for most simulations, which

typically do not include full simulation models for all of the interfaces and external devices. Future ACE releases may include more simulation options.
