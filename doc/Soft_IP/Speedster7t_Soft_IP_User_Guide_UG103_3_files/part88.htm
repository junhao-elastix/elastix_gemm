i_in_reg_rstn


Input

If "Enable Input Registers" is enabled, i_in_reg_rstn is the synchronous active-low reset for input registers.

If "Enable Input Registers" is not enabled, i_in_reg_rstn

doesn't exist.


i_pipeline_ce


Input

If "Pipeline registers" is selected (greater than 1), i_pipeline_ce

is the clock enable for pipeline and the accumulator registers.

If "Pipeline registers" is not selected (0), i_pipeline_ce does not exist.


Name

Direction

Description


i_pipeline_rstn


Input

If "Pipeline registers" is selected (greater than 1), i_pipeline_rstn is the synchronous active-low reset for pipeline and accumulator registers.

If "Pipeline registers" is not selected (0), i_pipeline_rstn does not exist.


i_load


Input

If "Accumulation" is enabled, i_load resets the accumulator to i_din_a × i_din_b , ignoring the previous value. This signal is internally pipelined to have the same latency as i_din_a × i_din_b .

If "Accumulation" is not enabled, i_load does not exist.

o_dout[(fp_size–1):0]

Output

Result of multiplication and optional accumulation.


o_status[1:0]


Output

Error status of o_dout :

2'b00 – normal.

2'b01 – result is ± 0.0.

2'b10 – result is ± infinity.

2'b11 – last operation had underflow. The result is ± 0.0.
