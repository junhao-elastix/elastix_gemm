‌Chapter 1 : Introduction‌


There are a number of available soft IP cores for the SpeedsterÂ®7t family of FPGAs. Each of these cores has an IP configurator within ACE that allows configuration of the soft IP core. When configured, the generated wrapper for the core can be instantiated within a user project enabling both synthesis and simulation of the design.

This document describes the available soft IP cores and the methods for configuration and instantiation of each. Soft IP cores are primarily implemented using the components present in the FPGA programmable fabric. For details of these components, refer to the Speedster7t IP Component Library User Guide (UG086).
