No


Yes, No

0 – No accumulation: ( dout_re + dout_im · j ) = ( i_din_a_re + i_din_a_im · j ) × ( i_din_b_re + i_din_b_im · j ).

1 – Accumulation: dout_re + dout_im · j is the accumulated value. The start of accumulation is signaled by asserting i_load = 1 .


Enable Input Registers


No


Yes, No

0 – No input registers.

1 – i_din_a_re , i_din_a_im , i_din_b_re , and i_din_b_im are registered. The input registers are controlled by the i_in_reg_a_ce , i_in_reg_b_ce , and

i_in_reg_rstn inputs. Enabling the input register adds one cycle of latency.
