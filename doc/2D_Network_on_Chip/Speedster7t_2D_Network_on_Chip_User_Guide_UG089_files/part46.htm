‌GDDR6‌

Addr[41:37] = 5'b00000

Addr[36:33] = Ctrl ID – Selects which of the sixteen GDDR6 channels the transaction is destined for: Addr[36:34] – Selects the controller

Addr[33] – Selects between the two channels on each controller

Addr[32:0] = Memory Address – The memory address for the specific controller and channel
