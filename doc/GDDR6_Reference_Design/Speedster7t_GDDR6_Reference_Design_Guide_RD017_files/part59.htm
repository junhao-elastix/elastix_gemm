‌Simulation Modes

This design supports the full-chip bus functional model (BFM) simulation mode using the VCS, Questa, or Riviera simulation tools. The standalone mode of simulation is supported using the VCS or Questa simulation tools. Check the simulation makefile for which modes are supported.
