acxip

In order for ACE to compile and place and route a design, it is necessary to use these source configuration

files to generate I/O ring constraint files. These I/O ring files specify the timing and placement constraints between the hardened subsystems (which form an I/O ring around the programmable fabric) and the fabric logic itself.

Further, the I/O ring files include the bitstream programming information for the hardened subsystems.
