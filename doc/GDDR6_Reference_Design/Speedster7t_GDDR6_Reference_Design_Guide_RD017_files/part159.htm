ACX_DEVICE_INSTALL_DIR

ACX_DEVICE_INSTALL_DIR

ACX_DEVICE_INSTALL_DIR

The optional environment variable


is used to select the DSM files. It should be set

to the path, including the base directory, of the device files within the DSM package.

When installed in ACE integration mode, the following setting should be used (with the Speedster7t

AC7t1500 FPGA as an example):



ACX_DEVICE_INSTALL_DIR = $ACE_INSTALL_DIR/system/data/AC7t1500


ACX_DEVICE_INSTALL_DIR = $ACE_INSTALL_DIR/system/data/AC7t1500


ACX_DEVICE_INSTALL_DIR = $ACE_INSTALL_DIR/system/data/AC7t1500

When installed as standalone, the following setting should be used, (with the Speedster7t AC7t1500 FPGA as an

example):



ACX_DEVICE_INSTALL_DIR = <location of standalone package>/system/data/AC7t1500

ACX_DEVICE_INSTALL_DIR = <location of standalone package>/system/data/AC7t1500

ACX_DEVICE_INSTALL_DIR = <location of standalone package>/system/data/AC7t1500
