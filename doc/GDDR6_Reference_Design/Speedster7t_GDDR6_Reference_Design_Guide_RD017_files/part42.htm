‌Top-Level Parameters

The GDDR6 reference design top-level RTL module has the following parameters:


‌Table 29 • Speedster7t AC7t1400 and AC7t1500 GDDR6 Reference Design Top-Level RTL Parameters


Parameter Name

Default Value

Description


GDDR6_NOC_CONFIG


8'b11111111

Signifies which of the 8 GDDR6 subsystems are enabled for 2D NoC interface testing (1 = enabled, 0 = disabled):

bit[0] – GDDR6_0

bit[1] – GDDR6_1

...

bit[7] – GDDR6_7


GDDR6_DCI_CONFIG


4'b1111

Signifies which of the 4 GDDR6 subsystems are enabled for DC interface testing (1 = enabled, 0 = disabled):

bit[0] – GDDR6_1 bit[1] – GDDR6_2 bit[2] – GDDR6_5 bit[7] – GDDR6_6
