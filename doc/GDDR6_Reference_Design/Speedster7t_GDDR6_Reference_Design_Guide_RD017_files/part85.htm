If using ACE 10.0 or later and standalone synthesis flow is desired as the default, then a modification is needed to

the

variable of the makefile. Change the

variable to a value other than "ace" to force the

standalone synthesis flow.




$> make run_ss


$> make run_ss


$> make run_ss


# Set $REV_DIR = "ace" for integrated ACE synthesis by default, ACE 10.0 onwards

# Set $REV_DIR to another value, i.e. rev_1, to use separate standalone synthesis by

default

# To override the default setting of REV_DIR, use the run_ss receipe REV_DIR := ace


Alternatively, call the make flow with the run_ss recipe.
