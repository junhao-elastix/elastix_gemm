ACE_INSTALL_DIR

ACE_INSTALL_DIR

In order to support relocatable projects, the designs make use of an environment variable, .

This variable should be set to point to the ACE installation directory (i.e., where the ace executable is installed). This variable is used by both synthesis and simulation to correctly locate the ACE library files. See Operating System for additional details.
