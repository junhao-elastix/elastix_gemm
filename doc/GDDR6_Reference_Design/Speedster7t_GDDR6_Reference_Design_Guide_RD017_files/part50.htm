The module instantiates a NAP_AXI_SLAVE/MASTER pair at column 9, row 2 of the fabric as the

default location. Therefore, it is important for VectorPath S7t-VG6/815 card fabric designs to not place any other

components in that region. However, if the design is not intended to provide the host access to the BW_BMC_IF module over the USB cable, then these NAPs can be placed anywhere within the fabric. The BW_BMC_IF module also requires another NAP_AXI_MASTER to be available in the fabric if the I2C initiator module is instantiated.
