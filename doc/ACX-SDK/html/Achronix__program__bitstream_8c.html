ACX_BITSTREAM_STATUS acx_program_bitstream ( ACX_DEV_PCIe_device *  device,
const char *  pciefilename,
uint8_t  encrypted 
)

Programs a stage1 PCIe bitstream via DMA. Requires a stage0 bitstream (.hex file) already loaded into flash or programmed via JTAG.

Parameters
devicePointer to an open PCIe device to program
pciefilenameThe stage1.pcie bitstream filename for your design
encryptedA boolean flag used to indicate if the bitstream is encrypted (1) or unencrypted (0)
Returns
ACX_BITSTREAM_STATUS_SUCCESS if successful
Note
This function will call all 'IP_block' initialization callbacks that are registered to the passed-in device if bitstream programming is successfully programmed
