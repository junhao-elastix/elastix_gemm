A statically defined global list of ACX_IP_LOCATION structs for each initialized IP block.

This array is populated src/Achronix_IP/Achronix_IP_config.c. You can modify the contents of the C file to change the default mapping for various IP blocks, such as those related to the DBI. You can also add mappings for your own custom IP blocks. Currently this information is static and will be built into the binary. In future SDK versions this information will be read in via meta data files
