Detailed Description

A union containing an anonymous struct describing the individual bitfields for the PCIe MSIX_pba_offset register.

Warning
The bits for the PBA_BIR and PBA_OFFSET register overlap, meaning that when trying to read the PBA_OFFSET field, the bottom 3 bits MUST be masked off. The PBA_BIR field has no special requirements and may be read as any struct member.
