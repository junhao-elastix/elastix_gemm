Detailed Description

This file defines enums, structures and functions for performing Direct Memory Access (DMA) transactions with a PCIe device.

Note
The following abbreviations are used in the function/variable/argument names:
  • buf == buffer
  • dev == device
  • addr == address
  • xfer == transfer
  • tactn == transaction
  • engn == engine
  • desc == descriptor
  • dir == direction
  • cntx == context
