Detailed Description

A helper structure for managing DMA buffer memory after it is allocated by the host.

Note
Use acx_dma_alloc_buff() to allocate a buffer, and acx_free_buf() to free one. Various higher level functions may also be used to allocate buffers, such as acx_dma_start_xfer_h2d() and acx_dma_start_xfer_d2h(). The virt_addr and data fields are read/write, all other fields should be considered read only by the application.
