Detailed Description

A union containing an anonymous struct describing the individual bitfields for the PCIe MSIX_table_offset register.

Warning
The bits for the BIR and TABLE_OFFSET register overlap, meaning that when trying to read the TABLE_OFFSET field, the bottom 3 bits MUST be masked off. The BIR field has no special requirements and may be read as any struct member.
