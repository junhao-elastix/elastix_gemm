Detailed Description

A helper structure for managing the data associated with descriptor list DMA transfers.

Note
A descriptor list node is a block of contiguous memory that consists of one or more data descriptors, and optionally ends with a link descriptor to the next node in the list. A list node is allocated as an ACX_DMA_buffer and then built on the host, and must then be transferred down to local memory on the device before it can be used to control a list transaction. Use acx_dma_alloc_list_node() to allocate a list node, and acx_dma_cleanup_list_node() to free one. All fields should be considered read only by the application.
