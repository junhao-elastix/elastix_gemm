Detailed Description

Holds DMA engine configuration data that is unlikely to change between DMA transfers.

Note
An engine takes a transaction and uses it to run a transfer. The engine should be configured once at the start of a program, and it can then be used with different transactions to perform multiple transfers. It is built using the acx_dma_build_engine_ctx() function, then used to initialize the engine using the acx_dma_init_engine_ctx() function.
