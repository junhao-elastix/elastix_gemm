Detailed Description

User space library interface for the Achronix PCIe driver.

Warning
This interface is normally intended for SDK Library internal use

◆ acxdev_interrupt()

int acxdev_interrupt ( acxdev_handle_t  handle,
int  index,
int  timeout 
)

Wait/Poll for MSI-X interrupt.

Parameters
[in]handlehandle to opened device
[in]indexinterrupt index
[in]timeout(timeout == 0) --> non-blocking (timeout > 0) --> block with timeout (units of msec) (timeout < 0) --> perform blocking release for this index
Returns
0 for success (interrupt received)
ENODATA if no interrupt present (i.e. non-blocking timeout == 0)
ETIME if timed out before interrupt present (i.e. blocked with timeout > 0)
ECANCELED if blocking released (i.e. blocked with timeout > 0 and another thread caused release)
EINTR if blocking released by a signal (i.e. blocked with timeout > 0 and a signal was received)
EALREADY if there is already a task blocked on this interrupt index
ENOENT if release requested (i.e. timeout < 0) and there is no other task pending on the interrupt index
